Instruções de Desenvolvimento: Site "Viagem no Tempo — Gabizinha"🎯 ObjetivoVocê é um Engenheiro de Frontend Sênior e Especialista em UI/UX Motion Design. Sua tarefa é desenvolver um site interativo de página única (SPA), totalmente responsivo e focado em dispositivos móveis (mobile-first), que sirva como um presente/surpresa de um irmão para sua irmã mais nova (Gabizinha).O site deve narrar uma história emocionante através de uma linha do tempo vertical com fotos dela desde bebê até o momento atual, culminando em uma mensagem afetiva com interações interativas.🛠️ Stack Tecnológica RecomendadaPara garantir o máximo de fluidez visual e facilidade de entrega, implemente usando:React (Componentização limpa)Tailwind CSS (Estilização responsiva e ágil)Framer Motion (Para animações baseadas em scroll e micro-interações físicas)Lucide React (Ícones modernos e leves)Canvas-Confetti (Opcional, para a interação final)🎨 Diretrizes Visuais (Do Design System)Você deve aplicar estritamente os seguintes tokens visuais:Fundo Principal (Cream Sand): #FFF3DD (Garante um tom nostálgico de papel antigo).Fundo de Destaques (Warm Canvas): #FFFAF1.Cor de Acento (Soft Rose): #FCA5A5 (Usado para botões, linha do tempo e corações).Texto Principal (Charcoal Night): #3D3D3D (Evitar o preto puro #000 para manter o visual suave).Texto Secundário (Muted Earth): #555555.Fontes:Principal (Títulos/Corpo): Arimo (ou Sans-serif moderna padrão).Secundária (Metadados/Datas/Tags): Azeret Mono (ou Monospace equivalente) para dar um ar moderno de "registro".🧱 Arquitetura de Seções & FuncionalidadesSeção 1: A Entrada (Hero Section)Estado Inicial:Uma bela foto de infância antiga de tela cheia (Background Cover) com um overlay de vinheta escura (radial-gradient(circle, transparent 20%, rgba(0,0,0,0.8) 100%)).No centro, uma tipografia de impacto: "Olha esse site legal que encontrei, Gabizinha!"Um botão pílula (border-radius: 9999px) em destaque: "Viajar no Tempo 🚀".A Transição (Trigger de Abertura):Ao clicar no botão, disparar uma animação sutil: a vinheta escurecida suaviza para opacidade de 0.2 (revelando a foto em sua totalidade) e a imagem de fundo faz um zoom-out suave de scale: 1.1 para 1.0.Realizar um scroll suave automático para o início da linha do tempo.Seção 2: A Linha do Tempo (Timeline Spine)O Eixo Central:Um traço vertical tracejado (dashed), centralizado na tela, usando a cor #FCA5A5 (2px de espessura).Círculos marcadores (12px) com efeito de pulso luminoso que acendem à medida que o usuário rola a tela e passa por eles.Cards de Memória (Estilo Polaroid):Pelo menos 3 momentos cronológicos:2010 (O Início): "Você era um pacotinho" (Foto dela bebê).2016 (Aventuras): "Parceiros de Crime" (Foto de infância brincando juntos).2024 (O Hoje): "Orgulho Irmão" (Foto recente dela).Visual Polaroid: Os cards devem ser brancos puro (#FFFFFF), com bordas generosas simulando fotos físicas, cantos levemente arredondados e rotações aleatórias muito sutis (ex: rotate-1 ou rotate--1) para parecer que foram jogadas em cima de uma mesa.Animação de Scroll (Scroll Reveal): Conforme a tela é rolada, os cards devem entrar de forma ultra-suave com um leve slide vertical (translateY: 40px para 0) e fade-in (opacity: 0 para 1), usando uma curva cubic-bezier(0.16, 1, 0.3, 1).Seção 3: O Fechamento (Grand Finale)Um container destacado que apresenta de forma altamente legível a frase filosófica:"Por mais que cresça, sempre caberá no nosso coração!"Um botão interativo flutuante ou central em formato de Coração (#FCA5A5).Ao clicar nesse coração, disparar um efeito visual de micro-interação: corações flutuantes subindo pela tela (gerados dinamicamente via CSS ou framer-motion) ou uma explosão de confetes em tons pastéis e rosa.📝 Requisitos Técnicos de Implementação para o ClaudeAo codificar o arquivo único (ex: App.jsx ou um arquivo HTML integrado):Mobile-First Real: Garanta que todo o espaçamento de padding lateral no mobile use no mínimo $16\text{px}$ e que os botões tenham uma área de toque confortável de pelo menos $48\text{px} \times 48\text{px}$.Imagens Resilientes: Use URLs de imagens confiáveis de alta qualidade do Unsplash/Vecteezy como placeholders bonitos (ex: fotos de irmãos brincando, bebês dormindo, etc.), de modo que o site pareça pronto para produção imediatamente.Framer Motion Avançado: Utilize motion.div com as propriedades initial, whileInView, e viewport={{ once: true, margin: "-100px" }} para que a animação da linha do tempo pareça reativa e fluida no celular enquanto o usuário desliza o dedão.Performance: Adicione transições CSS suaves para interações de toque (efeito de clique no botão com encolhimento sutil active:scale-95).🚀 Exemplo de Estrutura de Código EsperadaGere a estrutura da aplicação dividida de forma limpa, por exemplo:// Exemplo de componente de Card Polaroid altamente polido
const PolaroidCard = ({ year, title, description, image, rotate }) => {
  return (
    <motion.div 
      initial={{ opacity: 0, y: 50 }}
      whileInView={{ opacity: 1, y: 0 }}
      transition={{ duration: 0.8, ease: [0.16, 1, 0.3, 1] }}
      viewport={{ once: true }}
      className={`bg-white p-4 pb-8 rounded-2xl shadow-xl max-w-sm mx-auto ${rotate} transform transition-transform duration-300 hover:rotate-0`}
    >
      <div className="overflow-hidden rounded-lg aspect-square mb-4 border-[8px] border-white shadow-inner bg-gray-100">
        <img src={image} alt={title} className="w-full h-full object-cover" />
      </div>
      <div className="font-mono text-xs text-rose-400 mb-1 tracking-wider">{year}</div>
      <h3 className="font-sans font-bold text-lg text-neutral-800 mb-2">{title}</h3>
      <p className="font-sans text-sm text-neutral-600 leading-relaxed">{description}</p>
    </motion.div>
  );
};
